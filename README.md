# LedgerBySMD – Personal Expenses Tracker

> A simple command-line personal expense tracker built with Python that lets you log and view your expenses, with persistent storage using JSON.

---

## About

**LedgerBySMD** is a lightweight, command-line personal expense tracker built with Python. It allows you to easily add and view your daily expenses through a simple interactive menu. All expenses are saved locally in a JSON file, ensuring your data persists between sessions. Perfect for beginners learning Python file I/O and JSON data handling.

## Features

- Add expenses with a name and amount
- View all logged expenses in a numbered list
- Persistent storage via a local `expenses.json` file
- Simple and clean CLI interface

## Getting Started

### Prerequisites

- Python 3.x

### Installation

```bash
git clone https://github.com/shanuka-upendra/LedgerBySMD-PersonalExpensesTracker.git
cd LedgerBySMD-PersonalExpensesTracker
pip install -r requirements.txt
```

### Usage

```bash
python app.py
```

Follow the on-screen menu to add or view your expenses.
